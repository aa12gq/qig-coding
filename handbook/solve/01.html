    报错显示 digital envelope routines::unsupported 我们查看一下 node 版本

    
$ node -v
v18.14.1

    可以看到，node 版本是比较新的，该版本在上周发布，以此可见，最新 Nodejs 版本显然与 vuepress 不兼容

    # 解决办法

    首先我们需要将 Nodejs 降级处理，查看官方文档显示最高支持 v16 版本

        Node.js v16.19.0+

    # 安装 npx 或 nvm 版本管理器

    这里只展示 nvm 的解决办法，没接触过 nvm 和 npx 的可以了解这篇文章 https://juejin.cn/post/7000652162950758431

    现在，你应该能够正常使用 nvm 命令，尝试运行以下命令，安装 NodeJs 16.x 版本

    nvm install 16

    切换到 16 版本后重新运行 vuepress 项目即可
